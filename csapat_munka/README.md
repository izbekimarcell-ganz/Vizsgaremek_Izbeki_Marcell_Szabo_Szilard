# Csapatmunka

Ez a mappa a csapatmunka koordinálását segíti.

## Mappastruktúra

- `feladatok/` - Feladat lista, hozzárendelések, státusz követés
- `utemterv/` - Projekt ütemterv, mérföldkövek
- `megbeszeles_jegyzokonyvek/` - Meetingek jegyzőkönyvei

## Együttműködési alapelvek

### Komunikáció
- Rendszeres egyeztetések tartása
- Problémák azonnali jelzése
- Konstruktív visszajelzések adása

### Feladatok kezelése
- Egyértelműen definiált feladatok
- Határidők betartása
- Előrehaladás rendszeres jelzése

### Konfliktuskezelés
- Nyílt kommunikáció
- Kompromisszumkészség
- Közös célok szem előtt tartása

## Sablon fájlok

A mappában találhatóak sablon fájlok, amelyeket használhattok:
- Feladat template
- Meeting jegyzőkönyv template
- Heti beszámoló template
