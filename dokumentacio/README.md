# Dokumentáció

Ez a mappa tartalmazza a vizsgaremek összes dokumentációját.

## Mappastruktúra

- `tervezes/` - Projekt tervezési dokumentumok (követelmények, rendszerterv, stb.)
- `jelentesek/` - Haladási jelentések és státusz dokumentumok
- `prezentaciok/` - Prezentációs anyagok (védéshez, köztes bemutatókhoz)

## Fontos dokumentumok létrehozásához

1. Minden dokumentumot dátummal és verzióval láss el
2. Használj egyértelmű fájlneveket
3. Markdown (.md) formátumot preferáld a jobb olvashatóság érdekében
4. Tartsd naprakészen a változásokat
